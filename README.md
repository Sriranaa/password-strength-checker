# Password Strength Checker

A web-based tool that evaluates password strength using JavaScript and Regular Expressions.

## Features
- Real-time password validation
- Checks for uppercase, lowercase, numbers, and special characters
- Minimum password length validation
- Password strength indicator (Weak / Medium / Strong)
- Dynamic background color based on password strength
- Show / Hide password toggle

## Technologies Used
- HTML
- CSS
- JavaScript
- Regular Expressions

## Project Structure
password-strength-checker
│
├── index.html
├── style.css
├── script.js
└── README.md

## How it Works
The password is validated using JavaScript Regular Expressions.  
Each rule contributes to the strength calculation and updates the UI in real time.
